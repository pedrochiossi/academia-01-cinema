# academia-01-cinema

## About
This is the first project of the Webjump Frontend academic course. 

It consists of a single page application that lists movies and tv shows based on popularity and exibition, using data from [TMDb](https://www.themoviedb.org/).

The application also allows to add movies or tv shows to a favorites list.

All data is kept in localStorage.

## Technology used
* Javascript 
* CSS 
* HTML 
* [Axios](https://github.com/axios/axios)
* [Tiny-slider](https://github.com/ganlanyuan/tiny-slider)
